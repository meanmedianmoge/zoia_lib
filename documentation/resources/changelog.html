4.00 (October 8, 2024)

----------------------

- Sequencer Updates:

  - Sequencer CV Step Behaviour: sequencer step can now be controlled by a

    CV value. In sequencer options, change "behaviour" option to "cv step".

  - Sequencer CV Ramp Output: along with CV, gate, and ratchet, a new

    output type called CV Ramp has been added to the sequencer. It ramps

    linearly between CV step values.

  - Sequencer Key Input: You can now input cv values into the sequencer

    steps using a CV input. Adds two new CV inputs: key input note and

    key input gate. When key input gate is triggered, key input note is

    programmed into a sequencer step. The "key input" option determines

    which step is programmed:

    - key input: active: the active (or currently playing) step is programmed.

    - key input: selected: the selected step is programmed.

    - key input: increment: the selected step is programmed, and then the

        next step is selected. With this option, the "queue start" will

        reset the selected step.

  - When using the Sequencer Key Input on a ratchet output, a keyboard input

    of C will set the ratchet to 0, an input of C# will set the ratchet to 1,

    and so forth.

  - Sequencer Pages: you can now have a sequencer with more than 32 steps.

    Now instead of selecting how many steps you want the sequencer to have,

    you select how many visible steps you want the sequencer to have. There

    is a new option for selecting how many pages you want your sequencer to have.

    When you make connections to the step buttons, you are connecting to the first

    page of the first track.

    - to step through sequencer pages, select a step button, and then press page

      up and down while holding shift.

- Sampler Updates:

  - Mono and Stereo WAV Support: the sampler now supports both mono and stereo

    WAV files, with a new 'channels' option allowing you to select the number

    of channels.

  - Additional Bit Depths: 8, 16, 24, and 32-bit WAV files are now supported.

  - Optimized Waveform Generation: .WFP files are no longer generated at startup.

    Instead, a .WFP file is created when a sample is loaded, reducing unnecessary

    file creation.

  - Sample Rate Converter: While samples must still have a 48kHz sample rate for use in

    the sampler, a built-in sample rate converter now allows you to convert individual

    samples on the fly. For converting multiple files, batch conversion on a computer is

    still recommended.

  - Improved File Selection: The sampler's file selection menu now supports an unlimited

    number of nested folders and samples per folder, removing the previous limitation of

    64 files per folder.

  - Bug Fixes: Resolved many bugs and crashes that were present in version 3.0.

- Display Improvements: Increased the screen's refresh rate, resulting in faster and

  smoother scrolling.

- Increased the maximum size of FAT32 partition that is created by the Format SD option

  in the config menu to 128GB.

- Euroburo headphone volume is set to a default value upon each reboot.

- MIDI CC controller numbers 61-63 are now reserved for controlling button/stompswitch

  presses, releases, and rotary turns. This functionality applies only to messages sent

  on the MIDI channel set in the Config menu.

  - Added a CV mixer/attenuverter module. It's like the audio mixer but for CV

    signals, with a few extra features! The atten parameter for each channel

    controls the amount and polarity of the signal that is passed through to the

    output: 1.0 passes the full signal, 0.5 fully attenuates it, and 0.0 fully

    inverts it. The 'mode' option lets you select how these signals are summed

    at the output. Summing mode simply adds up all of the signals, clipping

    anything outside the +1/-1 CV range. Average mode divides each signal by

    the number of inputs before summing them at the output. This ensures that

    the output  doesn't get clipped. Try mixing multiple LFOs to create a wild

    new waveform, or blend a few sequencers to create a unique melody!

  - Added alternative screen views for the ADSR, audio mixer, and CV mixer

    modules. Each new view is unique to the module, and aims to show you more

    information about the module as a whole. You can access available

    alternative views using the shift + view buttons. When you save a patch

    with modules that have alternative views, they will get saved. For the ADSR

    module, the new view displays a classic ADSR curve. For the audio mixer

    and CV mixer, the view places a mixing console on the screen. Pressing the

    rotary encoder will toggle between viewing the full console or just a

    single channel.

    page number.
